# Using Java
To generates numbers from 1 to 100 with the following rules:
- number divisible by 3 with the word BIG
- number divisible by 5 with the word BANG
- If the number is divisible by 3 and 5, with BIG BANG
- The array generate and output to ‘output.json’ file 
## Requirements
- Java JDK 8 or above

## Steps to Run
1. Install Java (version 8 or above).
2. Save the code as `BigBang.java`.
3. Open a terminal in the folder where the file is located.
4. Compile program: javac BigBang.java
5. Run program: java Bigbang
6. Succefull run should see generated output.json file
